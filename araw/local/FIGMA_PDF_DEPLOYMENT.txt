STEP 2A: NGINX PDF VIEWER (15 minutes setup)
================================================================================

Option A: Direct PDF serving with simple viewer

1. Copy PDF to server:
   scp public/araw-dashboard-v3.1-mockup.pdf [user]@[server]:/tmp/

2. Install Nginx:
   sudo dnf install -y nginx

3. Create web directory:
   sudo mkdir -p /var/www/araw-preview
   sudo cp /tmp/araw-dashboard-v3.1-mockup.pdf /var/www/araw-preview/
   
4. Create simple HTML viewer:
   sudo nano /var/www/araw-preview/index.html

Paste this:

5. Set permissions:
   sudo chown -R nginx:nginx /var/www/araw-preview
   sudo chmod -R 755 /var/www/araw-preview

6. Configure Nginx:
   sudo nano /etc/nginx/conf.d/araw-preview.conf

Add:
server {
    listen 80;
    server_name _;
    
    root /var/www/araw-preview;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location ~* \.(pdf)$ {
        add_header Content-Type application/pdf;
        add_header Content-Disposition inline;
    }
}

7. Start Nginx:
   sudo nginx -t
   sudo systemctl start nginx
   sudo systemctl enable nginx
   
8. Configure firewall:
   sudo firewall-cmd --permanent --add-service=http
   sudo firewall-cmd --reload

9. Access:
   http://[server-ip]

DONE! Shows Figma mockup in browser with PDF viewer.

================================================================================
STEP 2B: SIMPLE STATIC FILE SERVING (10 minutes)
================================================================================

Even simpler - just serve the PDF directly:

1. Copy PDF to server (as above)
2. Setup Nginx (as above)
3. Skip the HTML wrapper
4. Configure Nginx to serve PDF at root:

server {
    listen 80;
    server_name _;
    
    root /var/www/araw-preview;
    
    location / {
        try_files $uri /araw-dashboard-v3.1-mockup.pdf;
        add_header Content-Type application/pdf;
    }
}

Access: http://[server-ip]
Opens PDF directly in browser.
