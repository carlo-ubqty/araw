## KEY FUNCTIONAL REQUIREMENTS FROM DISCUSSION DOCUMENT

### Data Toggle Behavior:
1. **NAP vs NDCIP switching:**
   - Same elements EXCEPT fund sourcing and sectors
   - NAP: 8 sectors
   - NDCIP: 5 sectors (some intersection with NAP, especially energy)

2. **Amount vs Projects toggle:**
   - Affects only the violet "Total Projects" card
   - Amount shows total investment
   - Projects shows total project count

3. **Ongoing vs Completed:**
   - Filter affects all data displays

### Chart-Specific Notes:
1. **Fund Source Chart:**
   - Only sectors from NAP/NDCIP Data selection are displayed

2. **GHG VS 2020 Baseline:**
   - REPLACES old "GHG Reduction (Actual vs Baseline)"
   - Shows sector-based breakdown

3. **Climate Impact Drivers Map:**
   - REPLACES "Vulnerability Index"
   - Uses NAP Climate Impact Drivers (Source: NICCDIES)
   - Shows "Nationwide" instead of project-specific locations
   - Heat map shows PSF funds only
   - CIDs can be visible or hidden (user preference via checkbox)
